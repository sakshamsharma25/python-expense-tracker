# 💰 Python Expense & Budget Tracker

A desktop GUI application built with Python and Tkinter to help users track their monthly budget, log expenses, and view remaining balances.

## Features
* **Interactive UI:** Built using standard Tkinter and TTK themes.
* **Local Storage:** Automatically saves all data to a local JSON file so you never lose your history.
* **Live Calculations:** Instantly updates your remaining budget and total spent as you add or delete items.
* **Category Filtering:** Sort your expense history by custom categories.

## How to Run
1. Ensure you have Python installed.
2. Clone this repository or download the `.py` file.
3. Run the script in your terminal: `python main.py`
